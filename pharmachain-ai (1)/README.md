PharmaChain is an AI + Blockchain powered system that reads handwritten prescriptions, extracts medicines automatically, and verifies drug authenticity on the Ethereum network.
It prevents counterfeit drugs, reduces prescription errors, and brings transparency to the pharmacy workflow.

Key Features

AI OCR for handwritten prescriptions

Medicine & dosage extraction

Suggests brand & generic alternatives

Ethereum-based batch verification

Counterfeit drug detection

Expiry & recall alerts

On-chain prescription hashing

3D interactive dashboard (React Three Fiber)

Tech Stack

Frontend: React, React Three Fiber, TailwindCSS, Three.js
Backend: Node.js / FastAPI
AI: OCR (TrOCR/Donut), NLP (NER models)
Blockchain: Solidity, Hardhat, Ethereum (Sepolia/Goerli)
Storage: PostgreSQL, IPFS (for metadata)
